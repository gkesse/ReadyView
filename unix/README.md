# Construction du Projet sous UNIX

La construction du projet sous UNIX nécessite les scripts suivants :

* Pour générer le fichier Makefile : **cmd_qmake.sh**
* Pour ouvrir l'IDE Qt Creator : **cmd_qtcreator.sh**
* Pour compiler le projet : **cmd_compile.sh**
* Pour exécuter le projet : **cmd_run.sh**
* Pour nettoyer le projet : **cmd_clean.sh**
